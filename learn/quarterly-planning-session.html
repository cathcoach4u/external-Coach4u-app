📅 Quarterly Planning Session

A half-day every 90 days to review the quarter and set the next quarter's priorities.

What this is

A half-day session every 90 days where the leadership team resets at the start of each quarter. Reviews the quarter just gone, refreshes the strategic doc, and sets the next quarter’s priorities.

Why it matters

Without a quarterly reset, you’ll spend Q2 still chasing Q1 priorities and Q3 still confused about what changed in Q2. The session is what keeps the operating system tight — the weekly meetings handle the tactical; the quarterly session handles the strategic.

Pre-work (1–2 weeks before)

  • Every leader completes the Team Check-in (17 statements, 1–5 scores)
  • Aggregate results circulated 48 hours before the session
  • Each leader reviews last quarter’s session notes + the 1-Year Plan
  • Bring: current numbers, calendar, notebook

The agenda (4 hours)

  1. Check-in — 15 min. Each leader: bests (personal + business), what’s working / not since last quarter, expectations for today. Capture “not working” items as issues.
  2. Review prior quarter — 45 min.
    • Walk last quarter’s goals one by one
    • Each owner: done / not done — what % completed
    • Target: 80% completion across all goals (you’ll never hit 100% if you’re stretching)
    • Under 80%? Dig into why before setting new goals
  3. Numbers review — 15 min. Year-to-date vs. the 1-Year Plan. What’s tracking? What’s slipping? Anything slipping 8+ weeks becomes an issue.
  4. One-Page Plan review — 45 min. Walk the strategic doc top-to-bottom. Update anything that’s changed since last quarter (Core Focus, Targets, Marketing, Team). Most quarters this is small edits; once a year (at the annual) it’s a bigger refresh.
  5. Build the issues list — 30 min. Brainstorm everything blocking progress. The 3 lowest-scoring statements from the team check-in are starting points. Don’t solve yet — just capture.
  6. Operating-system health check — 15 min. Rate each component 1–10:
    • Strategy clarity
    • Team accountability
    • Weekly meeting quality
    • Numbers / scorecard discipline
    • Issue resolution speed
    • Process documentation
    Anything ≤ 7 goes on the issues list.
  7. Set next quarter’s goals — 45 min. Run the Quarterly Goals exercise (see that guide): silent listing → Keep / Kill / Combine → SMART filter → first step + by-when. 3–7 company rocks. Then each leader picks 1–3 personal rocks.
  8. Discuss & Resolve key issues — 30 min. Work the top 3 issues using the 3-step protocol. Review One-Thing personal commitments from the last annual session — still on track?
  9. Conclude — 10 min.
    • Cascading messages: what does each leader tell their team after today?
    • Rate the session 1–10 with one sentence why

Common pitfalls

  • Treating it as a celebration / status meeting. The point is to make decisions.
  • Setting the same priorities every quarter. If a goal repeats unchanged, ask why it didn’t ship.
  • Skipping the operating-system health check. That’s where the meta-issues surface — the ones that won’t show up in goals or numbers.
  • Running it in the office where everyone drifts back to their desks at break. Block the room. Phones away.
  • Not circulating the Team Check-in results before. You’ll waste 90 min mining what the data already shows.
Coach’s tip
Schedule the next quarterly session at the end of this one. If you wait, it’ll slip into a busy quarter, then slip again. The whole operating system depends on this rhythm — protect the date like a board meeting.
